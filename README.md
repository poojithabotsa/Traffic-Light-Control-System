# Traffic-Light-Control-System
Traffic Light Control System with Pedestrian Counter

This project simulates a traffic light control system using Arduino Uno. The system uses Red, Yellow, and Green LEDs to represent traffic signals. An LCD displays the pedestrian count, and a push button is used to request pedestrian crossing.

Components Used

- Arduino Uno
- 16x2 I2C LCD
- Push Button
- Red LED
- Yellow LED
- Green LED
- Resistors
- Breadboard
- Jumper Wires

Working

1. Green light remains ON during normal traffic flow.
2. When the pedestrian button is pressed, the pedestrian count increases.
3. The signal changes from Green to Yellow.
4. Then Yellow changes to Red.
5. After pedestrian crossing time, the signal returns to Green.
